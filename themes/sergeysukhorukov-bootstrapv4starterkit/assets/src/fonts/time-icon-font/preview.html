Let's use your Icon Font!

Congratulations, your Icon Font miscellaneous-font from Orion Icon Library was created successfully!
In the downloaded folder, you will find the file called iconfont.css [Open] and the font files called miscellaneous-font.* in eot, svg, ttf, woff and woff2 formats.

Include Styles

Place the file iconfont.css in the same folder of the five font files and include the css in your html. 

<link href="path/to/iconfont.css" rel="stylesheet"/>

Otherwise include the snippet below in your own stylesheet, paying attention to point the font files url("./miscellaneous-font.*") to the right location. 

@font-face {
  font-family: "miscellaneous-font";
  src: url("./miscellaneous-font.eot");
  src: url("./miscellaneous-font.eot?#iefix") format("embedded-opentype"),
       url("./miscellaneous-font.woff2") format("woff2"),
       url("./miscellaneous-font.woff") format("woff"),
       url("./miscellaneous-font.ttf") format("truetype"),
       url("./miscellaneous-font.svg#miscellaneous-font") format("svg");
  font-weight: normal;
  font-style: normal;
}

@media screen and (-webkit-min-device-pixel-ratio:0) {
  @font-face {
    font-family: "miscellaneous-font";
    src: url("./miscellaneous-font.svg#miscellaneous-font") format("svg");
  }
}
